# Varieta Auto Peças - Landing Page Premium

## 🚀 Descrição

Landing Page premium e moderna para a empresa **Varieta Auto Peças**, desenvolvida com HTML5, CSS3 e JavaScript vanilla, inspirada em marcas como Apple, Porsche, BMW, Stripe e Linear.app.

## ✨ Características

### Design
- ✅ Minimalista e premium
- ✅ 100% responsivo (mobile-first)
- ✅ Paleta de cores sofisticada (azul escuro, cinza, branco)
- ✅ Tipografia elegante e bem espaçada
- ✅ Glassmorphism sutil
- ✅ Muitos espaços em branco (whitespace)
- ✅ Hierarquia visual perfeita

### Seções
1. **Navbar** - Fixa, com efeito blur ao scroll
2. **Hero** - Fullscreen com CTA estratégico
3. **About** - Sobre a empresa com features
4. **Products** - Grid de categorias com hover effects
5. **Brands** - Slider infinito de marcas parceiras
6. **Differentials** - Diferenciais da empresa
7. **Testimonials** - Depoimentos com glassmorphism
8. **CTA Section** - Call-to-action para WhatsApp
9. **Contact** - Formulário e informações de contato
10. **Footer** - Minimalista com links rápidos

## 🚀 Como Usar

1. Clone o repositório
2. Abra `index.html` em um navegador moderno
3. Configure os links de WhatsApp conforme necessário
4. Customize as cores, textos e imagens

## 📱 Responsividade

- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px
- **Mobile**: < 768px

Todos os componentes são 100% responsivos.

## 🎨 Customização

### Cores
Modifique as variáveis CSS em `styles/reset.css`:

```css
:root {
    --primary-dark: #1a1f3a;    /* Azul escuro */
    --accent-blue: #0066ff;     /* Azul vibrante */
    --white: #ffffff;           /* Branco */
    /* ... outras variáveis */
}
```

### Tipografia
Altera em `styles/main.css` ou customize via Google Fonts.

### Imagens
Substua as imagens de exemplo por suas próprias imagens.

### WhatsApp
Atualize os links em:
- Navbar
- Hero section
- CTA section
- Contact section
